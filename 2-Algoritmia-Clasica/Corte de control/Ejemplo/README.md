Deben tenerse en cuenta las siguientes precondiciones:
• El archivo se encuentra ordenado por provincia, ciudad, sucursal y
vendedor.
• Se debe informar el total de vendido en cada sucursal, ciudad y pro-
vincia, así como el total final.
• En diferentes provincias pueden existir ciudades con el mismo nom-
bre, o en diferentes ciudades pueden existir sucursales con igual
denominación.
