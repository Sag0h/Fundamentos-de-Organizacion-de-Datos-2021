Una empresa posee un archivo ordenado por código de pro-
motor con información de las ventas realizadas por cada uno de
ellos (código de promotor, nombre y monto de venta). Sabiendo
que en ese archivo pueden existir uno o más registros por cada
promotor, realice un procedimiento que reciba el archivo ante-
riormente mencionado y lo compacte (esto es, generar un nuevo
archivo donde cada promotor aparezca una única vez con sus
ventas totales).
